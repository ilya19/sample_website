Important HTML Tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell

Inline Elements

Tag Name Purpose Example Result
em emphasized make empasized text <em>technical sophistication</em> technical sophistication
strong strong important text <strong>important text</> important text
a anchor link to other web page <a href="http://example.com/">example.com</a> example.com
img image insert an image <img src="images/storm_trooper_tagged.jpg" />
code code display text in monspace font, useful for displaying code listings <code>SELECT * FROM users WHERE username = %1 AND password = %2;</code> SELECT * FROM users WHERE username = %1 AND password = %2;
